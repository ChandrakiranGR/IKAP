# Technique: v3_system_prompt

## Model
gpt-4o-mini

## User Query
I am a Northeastern student. I am using an android mobile and I am getting an error when I am trying to connect to NUWave.

## Response

Category: WiFi and network connectivity  
Clarifying question: What is the specific error message you are receiving when trying to connect to NUWave?  
Steps (general guidance, not official Northeastern instructions):  
1. Ensure that your mobile device's WiFi is turned on and airplane mode is disabled.  
2. Forget the NUWave network on your device by going to WiFi settings, selecting NUWave, and choosing "Forget."  
3. Restart your mobile device to refresh the network settings.  
4. Go back to the WiFi settings and search for available networks.  
5. Select NUWave and re-enter the password if prompted.  
6. If the issue persists, try moving closer to the router or access point.  
If this does not resolve your issue: Contact Northeastern IT Support and include:  
- Your device/OS  
- The step where the issue occurred  
- Any error message shown
